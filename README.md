# ✍️ Stack-Based Text Editor 
A Java-based text editing application that utilizes **Stack Data Structures** to manage text operations and history.

---

### 📝 Project Description
This project was developed to demonstrate the practical application of **Stacks (LIFO)** in software development. The editor allows users to perform basic text operations while maintaining a history of actions for undo and redo functionalities.

### 🚀 Features
- **Core Operations:** Insert, Delete, and View text.
- **Undo/Redo System:** Implemented using two separate stacks (Undo Stack and Redo Stack).
- **Efficiency:** Optimized for constant time complexity $O(1)$ for push and pop operations.
- **Interactive CLI:** Simple and clean command-line interface for user input.

### 🏗️ Data Structures Used
- **Primary Stack:** Stores the sequence of characters or strings.
- **History Stack:** Specifically designed to keep track of deleted items for restoration.

### 🛠️ Tech Stack
- **Language:** Java (JDK 8 or higher)
- **Concepts:** Data Structures (Stacks), Object-Oriented Programming (OOP).
